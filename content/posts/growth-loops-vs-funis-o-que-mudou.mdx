O funil de marketing é uma metáfora elegante: você coloca muita gente no topo e uma parte converte no final. TOFU (top of funnel), MOFU (middle), BOFU (bottom). Awareness → Consideração → Decisão.

O problema? Funis são lineares. E o crescimento real não é linear.

## O que é um Growth Loop?

Um Growth Loop é um sistema onde o output de um ciclo vira o input do próximo. O crescimento se auto-alimenta. Não é um funil que esgota — é uma roda que acelera.

Exemplo clássico: o loop viral do Dropbox. Usuário usa o produto → convida amigos para ganhar espaço → amigos experimentam o produto → convidam outros amigos. Cada usuário traz mais usuários. O crescimento é exponencial, não linear.

<Callout>Funis têm começo, meio e fim. Loops são contínuos — e cada ciclo fica mais forte que o anterior.</Callout>

## Os 4 tipos de Growth Loop

### 1. Loop Viral

Usuários convidam outros usuários. Funciona quando o produto tem valor de rede (fica melhor com mais usuários) ou quando o convite oferece benefício mútuo.

### 2. Loop de Conteúdo

Usuários criam conteúdo na plataforma → conteúdo é indexado pelo Google → novos usuários chegam → criam mais conteúdo. Tripadvisor, Reddit, YouTube são exemplos.

### 3. Loop de Performance (Paid)

Receita de usuários existentes financia aquisição de novos usuários. Funciona quando o LTV é previsível e o CAC é otimizado continuamente. É o loop mais comum em e-commerce.

### 4. Loop de Produto (PLG)

O produto em si gera aquisição. O usuário usa o produto → experimenta valor → promove naturalmente → outros adotam. Notion, Figma, Slack cresceram assim.

## Como identificar o loop da sua empresa

- Mapeie como seus melhores clientes chegaram até você
- Identifique qual comportamento deles trouxe outros clientes
- Encontre o "momento AHA" — quando o usuário percebe o valor pela primeira vez
- Construa o produto/processo para maximizar a chegada a esse momento
- Meça o loop: tempo de ciclo, taxa de conversão em cada etapa, coeficiente viral

## Funil e Loop: não é um ou outro

Na prática, a maioria das empresas usa ambos. O funil é útil para entender e otimizar etapas individuais. O loop é o modelo mental estratégico — ele define como o crescimento se sustenta no longo prazo.

A pergunta que você deveria estar fazendo não é "como otimizo meu funil?" mas sim: "qual é o loop de crescimento da minha empresa — e o que está impedindo ele de girar mais rápido?"
